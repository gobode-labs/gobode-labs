# Security Policy

## Our Commitment
At Gobode Labs, **Self-Defending Code** is our standard. 
We provide tools that are mathematically resilient 
to common attack vectors.

---

## Supported Versions
| Version | Supported |
| :--- | :--- |
| 1.0.x | Yes |
| < 1.0 | No |

---

## Reporting a Vulnerability
We value the work of security researchers. 
If you find a logical flaw or a security vulnerability:

1. **Do Not** open a public issue.
2. Email us at **[b4pr3y@gmail.com]**.
3. Provide a detailed description of the flaw.

### Our Response Time
We will respond within **48 hours** to coordinate a secure fix and disclosure.
